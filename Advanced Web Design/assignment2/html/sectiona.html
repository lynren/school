Q1: Convert the following loose HTML5 format into XML format. Mention the required changes as comments. Then write the code in the answer.

<html>
 <Body> <!-- in XML, all elements must be lowercase: <body> -->
  <!-- incorrect nesting with italic element. put closing tag inside p -->
  <p><italic> Welcome to my web site</p></italic>
  <!-- p element is not closed. It must be closed in XML -->
  <p>the site more often PLEASE!
 </body>
</html>

XML code:

<!-- XML declaration with version and encoding type -->
<?xml version="1.0" encoding="utf-8"?>
<html>
 <body>
  <p><italic>Welcome to my web site</italic></p>
  <p>the site more often PLEASE!</p>
 </body>
</html>

Q2: Do the minimum changes to rename the following files so they are compatible with Windows 7 and higher versions of Windows OS, mention the reason why they are not compatible with these operating systems.

a. 0360-270/advancedWebsiteDesign.txt

Windows does not allow the "/" character in a filename and is not case sensitive. We can replace the "/" with "-" and make all the letters lowercase. This gives us: 0360-270-advancedwebsitedesign.txt

b. ?Webdesign.html

Windows does not allow the "?" character in a file name and is not case sensitive. We can remove the "?" and make all the letters lowercase. This gives us: webdesign.html

c. <Hello>.html

Windows does not allow "<" and ">" characters in a filename and is not case sensitive. We can remove the "<" and ">" characters and make all the letters lowercase. This gives us: hello.html
